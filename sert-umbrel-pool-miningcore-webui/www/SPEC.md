# Спецификация Mining Pool Web UI (public-pool-btc.ru)

Все данные — **только в реальном времени** (без демо, без фейковых показаний). Источники: API пула и CoinGecko.

---

## 1. Верхняя шапка

- **В навигации только:** Home / язык (иконка земли, прозрачная) / Theme.
- **Убрать из шапки:** Blocks, Miners, Help, Pools.
- **Логотип** — слева от «public-pool-btc.ru» (как и раньше).
- **Язык:** выпадающий список (English / Русский), на кнопке отображается EN или RU. Кнопка — иконка глобуса (земли), прозрачная.
- **Тема:** выпадающий список (Dark / Light).

---

## 2. Главная страница (монеты)

- **Меню** Home/Blocks/Miners/Help/Pools на главной **удалено**.
- **Таблица пулов.** Колонки (все данные в реальном времени с API):

| Колонка | Описание |
|--------|----------|
| **Pool** | Название пула (BTC, BCH, BC2, XEC, FB и т.д.) + логотип монеты. |
| **Algorithm** | Алгоритм майнинга (SHA-256, SHA-256D, Scrypt и т.д.). |
| **Miners** | Количество майнеров, подключённых к пулу. |
| **Hashrate** | Хешрейт пула (TH/s, MH/s, GH/s, PH/s, EH/s — в реальном времени). |
| **Network Hashrate** | Сетевой хешрейт (H/s … EH/s), корректное отображение в реальном времени. |
| **Network Difficulty** | Сложность сети (в реальном времени). |
| **Current Price** | Текущая цена монеты (CoinGecko), в реальном времени. |
| **Reward** | Награда за блок (в криптовалюте и при возможности в USD), в реальном времени. |

- **Монеты:** BCH, BC2, XEC, FB (и другие, которые отдаёт API).

---

## 3. Страница монеты (например BCH, BC2, XEC, FB)

- **Логотип монеты** в заголовке.
- **Меню** Home / Blocks / Miners / Help / Pools — **оставить на внутренних страницах** (под шапкой).

### 3.1 Левая колонка

- **Pool Statistics:** Miners, Hashrate (в реальном времени).
- **Block Statistics:** Current Effort, Last Block, Blocks found (в реальном времени).
- **Network Statistics:** Network Difficulty, Network Hashrate, Current Height (в реальном времени).

### 3.2 Правая колонка

- **Current Price:** USD и % изменения (в реальном времени).
- **Price BTC:** в BTC и % (в реальном времени).
- **График хешрейта пула и сложности сети** за период (в реальном времени):
  - Ось: время; значения в G/s, M/s, PH/s, EH/s и т.д.
  - **Синяя линия** — Pool Hashrate (хешрейт пула по времени).
  - **Оранжевая линия** — Network Difficulty (сложность сети, в G и т.д.).
  - Оранжевая линия: заливка под кривой, плавные линии.
  - При наведении: вертикальная подсветка (в тёмной теме — светлая полоса, в светлой — тёмная); подписи/цифры через CSS-переменные (dark — белые, light — тёмные).
  - **Включить/выключить:** чекбоксы (или переключатели) для отображения «Pool Hashrate» и «Network Difficulty», чтобы можно было включать/выключать каждую линию и корректно отображать данные.

### 3.3 Ниже графика

- Блок: **Algorithm** / **Reward Scheme** / **Block Reward** / **Pool Fee** (все в реальном времени).
- **Your Wallet** — поле ввода кошелька и кнопка поиска (сохранены).
- **How to Connect** — кнопка ведёт на `#/help/{poolId}` (инструкция по подключению).

---

## 4. Воркеры — имя вместо адреса

- В списке майнеров и в дашборде майнера в колонке **Worker** выводится **имя воркера**, а не адрес:
  - из полей API: `workerName` / `worker` / `name`;
  - или часть после точки в `address.workerName` (например D0, D1, brain).
- Если имени нет — показывать адрес или «Default» (по согласованию).

---

## 5. Вкладка Miner (дашборд майнера)

- URL-пример: `#/miner/bc2/{address}/dashboard`.
- **Над вкладками:** Pool Wallet (адрес) + стрелка → кликабельно, ведёт в explorer.

### 5.1 Вкладки

- **Dashboard** / **Rewards** / **Payouts** — все рабочие.

### 5.2 Dashboard — блоки в реальном времени

- **Workers:** Online, Offline (количество).
- **Hashrate:** Current Hashrate (30m), Average Hashrate (1h) — в G/s, M/s, PH/s, EH/s.
- **Work:** Share Sum, Personal Effort (%), Blocks.
- **Reward:** Unconfirmed, Balance, Pending (все в реальном времени).

### 5.3 Hashrate Chart (график)

- Две линии в реальном времени:
  - **Синяя** — Current Hashrate (30m).
  - **Оранжевая** — Average Hashrate (1h), с заливкой и плавными линиями.
- Вертикальная подсветка при наведении (тёмная тема — светлая полоса, светлая — тёмная); подписи через переменные.
- **Включить/выключить:** переключатели для «Current Hashrate (30m)» и «Average Hashrate (1h)», чтобы обе линии корректно отображались и их можно было отключать.

### 5.4 Additional Information (под графиком)

- **Last Best Share** — в реальном времени (например, «X hours ago» или время).
- **Best Share** — в реальном времени (M, G, T и т.д.).
- **Network Difficulty** — в реальном времени (G и т.д.).
- **Port Difficulty** — в реальном времени (сложность порта подключения).

### 5.5 Таблица воркеров (Worker Statistics)

Все колонки заполняются **только реальными данными** с API:

| Колонка | Описание |
|--------|----------|
| **Worker** | Только имя воркера (не адрес): brain, D0, D1 и т.д. |
| **Hashrate (30m)** | Хешрейт за 30 минут (G/s, M/s, PH/s, EH/s). |
| **Hashrate (1h)** | Хешрейт за 1 час. |
| **Valids** | Количество действительных шар. |
| **Invalid** | Количество/процент неверных шар. |
| **Stale** | Количество/процент устаревших шар. |
| **Best Share** | Лучший шар (M, G, T). |
| **Port** | Порт, к которому подключён воркер (например «1M RU»). |
| **Last Share** | Время последнего шара (например «X minute ago»). |

- Графики и все перечисленные поля — **включены и работают в реальном времени** (без демо).

## 6. Справочные описания полей (как в ТЗ)

### 6.1 Таблица пулов (главная)

- **Pool (Пул):** Название и логотип монеты (BTC, BCH, BC2, XEC, FB и т.д.).
- **Algorithm:** Алгоритм (SHA-256, SHA-256D, Scrypt и т.д.).
- **Miners:** Количество подключённых майнеров.
- **Hashrate:** Вычислительная мощность пула (TH/s, MH/s, GH/s и т.д.).
- **Network Difficulty:** Сложность сети; влияет на скорость добычи.
- **Current Price:** Текущая цена криптовалюты.
- **Reward:** Награда за блок (в майнящейся валюте и при возможности в USD).

### 6.2 Страница монеты — график

- **Синяя линия:** Pool Hashrate — хешрейт пула по времени (PH/s и т.д.).
- **Оранжевая линия:** Network Difficulty — сложность сети (G и т.д.), с заливкой и плавными линиями.
- Вертикальная подсветка при наведении; подписи в dark/light через переменные.

### 6.3 Miner Dashboard

- **Workers:** Online / Offline.
- **Hashrate:** Current (30m), Average (1h) в G/s, M/s, PH/s, EH/s.
- **Work:** Share Sum, Personal Effort (%), Blocks.
- **Reward:** Unconfirmed, Balance, Pending.
- **График:** синяя — Current Hashrate (30m), оранжевая — Average Hashrate (1h), с заливкой и подсветкой.
- **Last Best Share, Best Share, Network Difficulty, Port Difficulty** — все активны и в реальном времени.
- **Таблица воркеров:** Worker (имя), Hashrate (30m), Hashrate (1h), Valids, Invalid, Stale, Best Share, Port, Last Share — все в реальном времени.
