# Иконки приложений

Иконки в этой папке используются в `umbrel-app.yml` приложений магазина, чтобы они гарантированно отображались в Umbrel (ссылки на ваш репозиторий).

- **pool-dashboard.svg** — для приложения «Моя панель — Pool Dashboard» (sert-umbrel-pool-poolui).
- **webui-native.svg** — для приложения «MiningCore Web UI (родная)».

В манифестах указывайте URL вида:
`https://raw.githubusercontent.com/Sert1985n/umbrel-pool-app-store/main/icons/имя-файла.svg`
