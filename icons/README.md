# Иконки приложений

Основной источник иконок: **[cryptologos.cc](https://cryptologos.cc/)** — в `umbrel-app.yml` указаны URL вида `https://cryptologos.cc/logos/{монета}-{символ}-logo.svg` (например bitcoin-btc-logo.svg, litecoin-ltc-logo.svg).

Дополнительно в этой папке:
- **pool-dashboard.svg** — запасная иконка для «Моя панель — Pool Dashboard».
- **webui-native.svg** — запасная для «MiningCore Web UI (родная)».

Если иконка с cryptologos.cc не грузится, в манифесте можно подставить:  
`https://raw.githubusercontent.com/Sert1985n/umbrel-pool-app-store/main/icons/имя-файла.svg`
