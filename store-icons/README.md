# Store icons (local only)

All icons for Umbrel Pool app store are stored here. No external URLs.

- **miningcore.png**, **webui.png**, **poolui.png** — main apps
- **dozzle.png**, **postgres.png** — optional apps
- **&lt;coin&gt;.png** (btc, bch, xec, dgb, …) — node app icons
- **generic.png** — fallback

Manifests use: `https://raw.githubusercontent.com/Sert1985n/umbrel-pool-app-store/main/store-icons/<name>.png`
