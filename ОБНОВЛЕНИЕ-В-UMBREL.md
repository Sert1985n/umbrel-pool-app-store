# Обновления не видны в Umbrel — что делать

Umbrel подтягивает community store с GitHub не сразу и не всегда показывает «доступно обновление». Сделай по шагам:

## 1. Подожди 5–10 минут
После пуша в GitHub Umbrel может обновить магазин в течение **5–10 минут** (интервал подтягивания). Ничего не нажимай — просто подожди и снова зайди в магазин Umbrel Pool.

## 2. Обнови магазин вручную
- **App Store** → открой магазин **Umbrel Pool**
- Найди кнопку **Update** / **Refresh** / **Обновить** (часто вверху страницы магазина или в меню) и нажми
- Зайди в **SOLO Crypto Mining** и открой карточку **Pool Dashboard (моя панель)** — в подзаголовке должно быть **(v1.0.2)**. Если видишь версию — магазин обновился.

## 3. Перезагрузи Umbrel
Если через 10 минут ничего не изменилось:
- Перезагрузи устройство с Umbrel (или перезапусти сервис Umbrel)
- Снова зайди в App Store → Umbrel Pool и проверь карточку Pool Dashboard (должно быть v1.0.2 в tagline).

## 4. Крайний вариант — заново добавить магазин
Если по-прежнему старый контент:
- **Settings** → **Community App Stores** → удали **Umbrel Pool**
- Добавь снова: `https://github.com/Sert1985n/umbrel-pool-app-store`
- Зайди в магазин и проверь приложения — версии и текст должны быть новые.

---

**Как проверить, что обновление подтянулось:** на карточке **Pool Dashboard (моя панель)** в подзаголовке должно быть написано **(v1.0.2)**. Если там старая строка без версии — Umbrel ещё не обновил кэш.
