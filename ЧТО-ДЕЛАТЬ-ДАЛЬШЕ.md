## Шаг 1. Отправить изменения в GitHub (если ещё не отправил)

На своём компьютере в папке проекта выполни в терминале:

```bash
cd C:\Users\WIN-10\Desktop\Umbrel-Pool-Project
git add .
git status
git commit -m "Панель 8563: только наши 28 монет; дашборд 8561: подсказка при пустом списке пулов"
git push origin main
```

Если `git` не настроен или пушишь в другой репозиторий — просто сохрани файлы и при необходимости залей изменения на GitHub вручную (через сайт или GitHub Desktop).

## Шаг 3. Чтобы на 8561 появились монеты (таблица не пустая)

Таблица на **http://ТВОЙ-IP:8561** заполняется только тогда, когда в MiningCore настроены пулы. Сделай по порядку:

1. **Залить наш список монет на сервер** (один раз по SSH на Umbrel):

   ```bash
   curl -sSL -o /home/umbrel/.miningcore/coins.json "https://raw.githubusercontent.com/Sert1985n/umbrel-pool-app-store/main/templates/coins.json"
   ```

2. **Перезапустить MiningCore и панель Pool Config:**

   ```bash
   docker restart sert-umbrel-pool-miningcore_server_1
   sleep 15
   docker restart sert-umbrel-pool-pool-config_web_1
   ```

3. **Открыть в браузере:**  
   **http://ТВОЙ-IP:8562** → зайти в **Pool Configuration**.

4. В этой панели нажать по очереди:
   - **Refresh Master Coin List** — подтянутся наши монеты из `coins.json`;
   - **Setup Database Schema** — если ещё не делал;
   - **Generate Pool Config File** — сгенерируется конфиг с пулами.

5. В сгенерированном конфиге заменить плейсхолдеры **"xxx"** на свои кошельки (адреса для выплат). Сохранить конфиг и при необходимости перезапустить MiningCore.

6. Обновить страницу **http://ТВОЙ-IP:8561** — в таблице должны появиться пулы.

## Шаг 4. Панель на 8563 (наши 28 монет + настройка)

- Открой **http://ТВОЙ-IP:8563**.
- Сверху — список наших 28 монет и кнопка **Refresh Master Coin List**.
- **Ниже на той же странице** встроена панель настройки (как на 8562): в ней делайте **Setup Database Schema**, **Refresh Master Coin List**, **Generate Pool Config File**, создание кошельков и замену «xxx» на ваши адреса.
- Если встроенная панель не загружается — открой **8562** в новой вкладке по ссылке на странице.
