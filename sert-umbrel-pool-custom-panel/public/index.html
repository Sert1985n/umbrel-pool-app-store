Как работает Pool Configuration. Кнопка «Setup Database Schema / Generate Pool Config File» открывает MiningCore Web UI на порту 8562 — там создаёте кошельки, подставляете адреса вместо «xxx» и генерируете config.json. Монеты для настройки берутся из списка выше (наш coins.json). После обновления coins.json на сервере нажмите «Refresh Master Coin List» здесь и в 8562.
